Inside a Ford SYNC by Microsoft module

Thursday, December 31, 2009

 

The picture above is a Ford Sync by Microsoft module (basically an embedded computer).  I was curious, so I opened it up to see what was inside - here is a picture I took of the motherboard - the labelled components are described below:



1. Power & RS-232 port

2. Phillips high-sped Controller Area Network (CAN) transceivers

3. Maxim RS-232 controller

4. Mini USB ports

5. Freescale i.MX31 CPU (ARM1136JF-S RISC @ 400MHz, 128KB L2 cache, Vector Floating Point coprocessor, 473-pin BGA packaging)

6. Cirrus Logic Digital-to-Analog / Analog-to-Digital converter

7. Freescale internal voltage regulator, clock generator, and bus controller

8. Micron DDR2 SDRAM (64MB)

9. Texas Instruments low-dropout voltage regulator

10. Cambridge Silicon Bluetooth Radio, Baseband IC, and ROM

11. Samsung NAND Flash (256MB) which holds Windows CE: Automobile 5 OS

The one neat thing I noticed about each of these components when looking up their specs at the vendors website is that they are usually listed as “military grade” or “automotive grade”, which means that they were manufactured to operating in extreme heat/cold (the actual temperature limits listed on the Freescale site were quite impressive).  Also, the motherboard is extremely thick for the same reason.


Oh, and in case you are wondering - the module still worked after I put it back together again :-)

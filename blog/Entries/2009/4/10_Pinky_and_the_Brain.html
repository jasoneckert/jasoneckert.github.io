Pinky and the Brain

Friday, April 10, 2009

 

We have two Linux file servers in our classroom: Pinky and the Brain.


Pinky is a Dell P4 3GHz running Ubuntu Linux, while Brain is a quad-UltraSPARC Sun Enterprise 450 with lots of SCSI drives and RAM running Red Hat Linux (see above picture).


For those who don’t know about the animated TV show Pinky and the Brain, the Brain looks and sounds like and evil Orson Welles, and Pinky is “happy-go-lucky” with a cockney accent.  Each episode began with the following two lines:

  1. Pinky: "Gee Brain, what do you want to do tonight?"

  2. The Brain: "The same thing we do every night, Pinky—try to take over the world."


As a result, if you look in the /etc/rc.d/rc.local script on Brain, you will see the following lines (the script is run every morning when we boot Brain):


  1. if [ -x /bin/takeovertheworld ]

  2. then

  3. /bin/takeovertheworld

  4. fi


This script checks to see if the /bin/takeovertheworld program exists and has execute permission.  If it does, it will be executed.  If it does not, we’ll have to wait until tomorrow and then try it again.....

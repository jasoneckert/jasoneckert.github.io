The coolest arcade game ever

Tuesday, October 1, 2013

 

....is definitely Namco’s Ms. Pac-man/Galaga 20th Reunion (Class of ’81) released in 2001. The picture at the top of this post is the actual game in the movie Sharknado (2013).


So what makes this the best arcade game ever made?

Firstly, it has the three killer arcade games from the early 1980s: Ms. Pac-man, Galaga, and the original Pac-man (which you can play if you enter a secret joystick code described below).  These three games have stood the test of time - they were crazy addictive back in the early 1980s and people still go crazy over playing them today.  This was due, in part, to the fact that all three games were prime examples of Nolan Bushnell’s Law:  A good game should be easy to learn and difficult to master - it should reward the first quarter and the hundredth.


But this alone isn’t enough to make this the coolest arcade game ever. 


Remember that video game arcades died long ago.  You may only find one or two at the odd bar, laundromat, restaurant or pizza parlor.  But there is a very good chance that those machines are ones that people would be more likely to spend a quarter on, and Ms. Pac-man and Galaga were the two most successful Namco video games within North America!  Think of your local area - can you identify some place that still has an original Ms. Pac-man or Galaga game?  In my area (Kitchener-Waterloo), I can think of 2 off the top of my head: Sonny’s burger joint in Waterloo, and the Frederick Twin Cinemas in Kitchener.  Both places still have those arcades because people still put quarters into them!


So, let’s think like Namco - it’s the late 1990s and we want to release a brand new dual Ms. Pac-man/Galaga game because they are sure-fire winners (with a hidden Pac-man game in there for the Pac-man die hards).  How will we convince people to buy them?  In other words, how would Namco convince a restaurant owner that if they were to buy this arcade machine, they would be guaranteed to make their money back on it plus a lot more?  Simple.  They added a continue feature! This is pure genius on Namco’s part since old arcade games didn’t allow you to continue - instead, you only had 3 lives, and that was it, which meant you had to practice to get your name on the high score list.  With Ms. Pac-man/Galaga 20th Reunion (Class of ’81), if people want to get their name on a high score list, they just need to have enough $$ because they could insert another credit and continue from where they left off!  Even if the owner charged a quarter per play, people would continue several times, dropping a few dollars into the game and having loads of fun.  It was a complete success - people went nuts over these machines, and still do.  I’ve seen them all over the place, and they have become an easy-to-recognize icon.


Another important feature about this arcade is that it runs the exact same game code as the original games from the early 1980s.  But how is this even possible given that the hardware from back then went obsolete long ago?!?!?!?  Well, Namco simply ensured that their new board ran the same Zilog Z80 CPU platform!   Pac-man and Ms. Pac-man had a single Zilog Z80 CPU that ran at about 3MHz, and Galaga had three Zilog Z80 CPUs that ran at 3MHz each.  The new printed circuit board (PCB) that Namco made for this arcade has a Zilog Z180 CPU running at 20MHz to handle all of the game code (the Z180 is just a beefed up Z80).  The firmware and game programs themselves are are stored on two modern ROM (Read Only Memory) chips that have a much higher capacity, and the dozens of chips on the original PCBs that provided memory storage, input, sound and graphic generation are now performed by two custom logic chips that can easily perform all of the functions! 


And the whole board is tiny - the picture below shows the PCB next to the manual (standard 8.5x11” printed paper).  Modern arcade machines (after about 1987) started using a standard connector called JAMMA (Japanese Amusement Machinery Manufacturer’s Association) that allowed you to easily put different game PCBs in arcade cabinets without having to modify the wiring.  The PCB below has a JAMMA connection that essentially allows you to stick this in any *relatively* modern arcade machine (which is what I’ve done ;-)



Finally, Namco decided to release the cabinet in three sizes (shown in the picture below) to fit the needs of different people and places: 

  1. •Full-sized - An upright cabinet that looked very similar to an old Ms. Pac-man or Galaga full-sized cabinet.....but with a huge 25” monitor (the original Ms. Pac-man and Galaga arcade games only had a 19” monitor).

  2. •Cabaret - A slimmer, shorter version of the full-sized upright that was marketed for home use only.  It had a 19” monitor, but instead of having a coin door, it had a sticker that looked like a coin door.

  3. •Cocktail - A two-player, table-based version with a 19” monitor that was very common in bars, pizza parlors, and pretty much everywhere else.  Namco probably sold as many of these as they did the Full-sized cabinets!



OK - so what are the secret codes I can enter if I find one of these arcades?


Recall from earlier that one of the coolest features of this game is that it also contains the original Pac-man game!  But the game is hidden from the start screen - instead, you need to enter a secret joystick code in order to play it.  When you insert your coin and get to the start screen shown to the right, simply enter the following joystick code:

  1. •Up,Up,Up

  2. •Down,Down,Down

  3. •Left, Right, Left, Right, Left

You’ll hear a chime at this point and the Pac-man ghost on the screen will change to a different color - if you hit the Ms. Pac-man start button now, you’ll now be playing good old Pac-man game instead of Ms. Pac-man!  


Similarly, if you want to enable fast speed in Ms. Pac-man or fast shooting in Galaga, simply perform the following pattern after inserting your coin: 

  1. •Left, Right, Left, Right

  2. •Up, Up, Up

  3. •Fire button

Again, you’ll hear a chime at this point, and if you start Ms. Pac-man, it will be the rapid version - if you instead start Galaga, you’ll have rapid fire!  One thing to note is that these codes only affect the current game - you’ll need to enter these code each time you want to start Pac-man, or enable fast speed/shooting.


Play Galaga for free (for a short time)

Without entering any money, just watch the demo screen until it cycles to the Galaga demo.  During the Galaga demo, when the boss Galaga starts its tractor beam to suck up the ship, you can move the joystick and actually take control of the game!  You can play the game for a single round, and if you shoot all of the boss Galaga on the screen before they try to suck up your ship, the game will reset and display ROM version information before returning to the normal start screen.  This is a great way to get the version of the ROMs used on the game board without opening the cabinet. 


Are there sequels to this arcade?

Yes. Because it was so incredibly popular, Namco released another version in 2005 called Pac-man 25th Anniversary, which contained the same games, but you could select Pac-man from the menu screen instead of entering a secret code.  The PCB for this game was almost identical too - however, it had newer and fewer components.  The cabinet was also released in the same three sizes:  full-sized (shown left), cabaret and cocktail.


In 2010, Namco released yet another arcade called Pac-man’s Arcade Party that contained  13 games in total: Pac-man, Ms. Pac-man, Galaga, Dig Dug, Galaxian, Pac-mania, Galaga '88, Mappy, Xevious, Rolling Thunder, Dragon Spirit, Rally-X, and Bosconian. However, the hardware for this system was not Zilog Z80-based - instead, it was basically the same hardware you’d find in a Sony Playstation 2.  As a result, Namco would have had to rewrite much of the code to work on this new platform, or run the original code in a software emulator on the new platform (much more likely). 
The gameplay for each game is identical to the original game, but the system loses a bit of it’s retro charm by ditching the Zilog Z80 platform.....it’s not longer an “authentic” original game from a technology standpoint.  Namco also released this version in the same three sizes (full-sized, cabaret & cocktail), with LCD monitors in place of the old CRT picture tube monitors (which are no longer manufactured in North America).  The picture to the right is a full-sized Pac-man’s Arcade Party cabinet.


I think it’s reasonable to assume that Namco will release another version in 2015, 2020, 2025, and so on.  After all, people still play these games, and places still buy them! 


Hopefully the price stays the same (or lower) given that the hardware components will only get cheaper over time.  This will allow the places that buy them to keep the games at 25¢ - these games are a lot of fun, but I can’t see people paying $1 to play or continue their games.  At 25¢ though, I’m pretty sure that it will be crazy popular for many years to come ;-)
